# Guia para agentes de IA

Este repositorio usa **spec-as-source**. Antes de editar codigo:

1. Leer `spec/norte-proyecto.md`.
2. Cargar `skill/orquestacion-spec-as-source/SKILL.md` si la solicitud es nueva.
3. Crear o abrir el plan en `spec/planes/`.
4. Ejecutar con `skill/ejecutar-plan/SKILL.md` por etapas.

No modificar `app/` sin plan activo. Aplicar `spec/checklists/testing.md` y `spec/checklists/seguridad.md` en cada etapa.
